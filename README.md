# Таблица товаров

#### Модели
+ Категории
  * имя
  * описание
  * активен
+ Продукты
  * имя
  * описание
  * аватар
  * цена
  * активен
  * категория
  * дата производства
  * дата создания
  * дата обновления
+ Контакт
  * инн
  * страна
  * адрес

#### Заполнение БД
  * *data.json* - общий дамп таблиц
  * команда ``seed`` - сидирование из данных команды
  * команда ``seed_json`` - сидирование из дампа
  * команда ``contact_seed`` - сидирование таблицы контактов
